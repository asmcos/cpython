# 列表

```
#coding:utf-8

l1 = [1,2,3,4]

'''
计算机的排序是从0开始
打印列表第三个元素
'''
print(l1[3])

'''
打印列表第0个元素
'''
print(l1[0])


l2 = ['abc','1','hello',1]

'''
列表是可以存放字符串，字符，和int类型的
'''

print(l2[0])

'''
-1是指最后一个，字符章节演示过
'''
print(l2[-1])

```
